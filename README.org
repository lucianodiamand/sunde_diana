* Proyecto Sunde Diana (i.MX53)

** Objetivo
Construir un cliente fino basado en i.MX53 con el siguiente flujo:

1. Detección y funcionamiento de NAND en el núcleo
2. Salida HDMI funcional en el núcleo
3. Sistema mínimo (Buildroot)
4. Cliente remoto (X2Go / RDP)
5. GPIO (LED y Switch para debug/control)

* Fase 1: Detección de NAND

** Objetivo
Que el núcleo detecte y pueda interactuar con la NAND

** Tareas

*** Soporte en kernel
- [ ] Habilitar MTD:
  - CONFIG_MTD
  - CONFIG_MTD_NAND
  - Driver NAND i.MX53

*** Verificación
- [ ] Revisar:
#+begin_src
dmesg | grep -i nand
cat /proc/mtd
#+end_src

*** Herramientas
- [ ] Incluir mtd-utils

*** Pruebas básicas
- [ ] Leer NAND:
#+begin_src
nanddump /dev/mtd0
#+end_src

- [ ] Borrar NAND (cuidado, probar con paginas al final del dispositivo):
#+begin_src
flash_erase /dev/mtd7 0 0
#+end_src

*** Particionado
- [ ] Definir un nuevo particionado:
  - mtd0: u-boot
  - mtd1: kernel
  - mtd2: rootfs

*** Filesystem
- [ ] Migrar a UBIFS

*** U-Boot

- [ ] Configurar:
  - bootcmd
  - bootargs

Ejemplo:
#+begin_src
console=ttymxc0,115200 root=ubi0:rootfs rootfstype=ubifs
#+end_src

- [ ] Boot desde NAND funcionando

* Fase 5: GPIO (LED y Switch)

** Objetivo
Debug y control del sistema

** LED

*** Identificación

- [ ] Encontrar GPIO asociado

*** Test

#+begin_src
echo <N> > /sys/class/gpio/export
echo out > /sys/class/gpio/gpio<N>/direction
echo 1 > /sys/class/gpio/gpio<N>/value
#+end_src

*** Heartbeat

- [ ] Script simple:

#+begin_src sh
while true; do
  echo 1 > /sys/class/gpio/gpio<N>/value
  sleep 0.5
  echo 0 > /sys/class/gpio/gpio<N>/value
  sleep 0.5
done
#+end_src

*** Mejor opción (kernel)

- [ ] Device tree:

#+begin_src dts
leds {
    compatible = "gpio-leds";
    heartbeat {
        gpios = <&gpioX Y GPIO_ACTIVE_HIGH>;
        linux,default-trigger = "heartbeat";
    };
};
#+end_src

** Switch

*** Lectura

#+begin_src
echo <N> > /sys/class/gpio/export
echo in > /sys/class/gpio/gpio<N>/direction
cat /sys/class/gpio/gpio<N>/value
#+end_src

*** Uso en boot

- [ ] Modo recovery:

#+begin_src sh
if [ "$(cat /sys/class/gpio/gpio<N>/value)" = "1" ]; then
  /bin/sh
fi
#+end_src

* Notas

- Mantener rootfs lo más chico posible
- Minimizar escrituras en NAND
- Usar LED como debug temprano

- Escritorio recomendado:
  - i3 (liviano y predecible)

- Alternativa extrema:
  - Sin window manager (solo X + RDP)

- RDP:
  - Más fácil de integrar en Buildroot

- X2Go:
  - Mejor rendimiento en redes lentas
  - Más complejo de integrar en cliente embebido
